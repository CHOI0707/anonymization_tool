# anonymization_tool
Anonymization tool to obfuscate images in dataset for preserving privacy

### Execution
python ./src/anonymization_tool.py -i ./input -o ./output -m ./face_model/face.pb -t 0.4  

### Requirements
python==3.8.18  
numpy==1.19.5  
tensorflow==2.3.0  
opencv==4.6.0
